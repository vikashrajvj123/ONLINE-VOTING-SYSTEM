# ONLINE-VOTING-SYSTEM
Project: Online Voting System in PHP with source code  Online Voting System is developed using PHP, CSS, and JavaScript. The project is an interesting, useful project. This project contains the admin side and user side where a user can vote for their favorite candidate. While the admin can add candidates, see voting results, and so on. About System Talking about the features of this system, the admin can manage the candidates, view results, and check feedback. While the user can simply vote for all the available candidates by giving some Basic Details of oneself. This system is A Interactive Way To Solve Conventional Voting. The voters can also easily vote for their favorite person without having to go to the voting area. Also, the admin can easily view voting status.
